
# Creating Sidebar Navigation

The navigation structure is controlled entirely by a file named `SUMMARY.md` in your GitHub repository.

## Defining Sections
Use H2 headers (##) in the SUMMARY file to create the bold category headings in the sidebar.

## Adding Links
Use a bulleted list format: `* [Title](path/to/file.md)`.

## Creating Nesting
To nest a page, simply indent the bullet point using two or four spaces under the parent page.
